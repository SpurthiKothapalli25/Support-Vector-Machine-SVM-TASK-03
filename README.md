#Cat vs Dog Image Classification Using Support Vector Machine (SVM)

## 📌 Problem Statement

The objective of this project is to implement a Support Vector Machine (SVM) model to classify images of cats and dogs. The model is trained on image data and learns to distinguish between the two classes based on extracted image features.

## 📂 Dataset

The project utilizes a Cats and Dogs image dataset containing labeled images of both categories. The dataset is used for training and evaluating the classification model.

## 🔄 Methodology

1. Data Collection and Loading
2. Image Preprocessing
3. Feature Extraction (HOG / Flattening)
4. Train-Test Split
5. SVM Model Training
6. Model Evaluation
7. Prediction and Classification

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-learn
* Matplotlib

## 🤖 Machine Learning Model

**Support Vector Machine (SVM)**

The SVM algorithm is used to classify images into two categories: Cat and Dog. It identifies an optimal decision boundary to separate the image features of both classes.

## 📊 Results

The trained model is evaluated using classification metrics and accuracy scores to measure its performance on unseen test data.
